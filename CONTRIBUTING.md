# Contributing to Refinish

Refinish is in early development. Thanks for your interest in contributing!

## Getting Started

1. Install [Node.js](https://nodejs.org) (v18+) and [Rust](https://www.rust-lang.org/tools/install)
2. Clone the repo and run:

```bash
npm install
npm run tauri dev
```

## Before You Submit a PR

Please **open an issue first** to discuss what you'd like to change. This helps avoid duplicate work and ensures your contribution aligns with the project direction.

## Development Notes

- The frontend is React + TypeScript with Tailwind CSS
- The backend uses Rust via Tauri 2
- Run `npm run tauri dev` for the development server with hot reload

## Code of Conduct

Be kind and constructive. We're all here to build something useful.
